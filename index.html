О проекте

Нейросиноптик

Программа, которая даёт точные прогнозы погоды.

Я написал её на языке Python используя библиотеку PyWeather. Общая длина кода измеряется сотнями строк.

На картинках справа можно увидеть пример кода и работы программы.

Как это работает?

  1. Пользователь делает запрос

    Для этого мы создали
    удобный интерфейс
  2. Программа использует
    специальные библиотеки

    PyWeather — библиотека,
    которая даёт доступ к данным
    о погоде.
  3. Запрашивает данные
    с других серверов

    Программа запрашивает данные
    у тех, кто профессионально
    занимается прогнозом погоды
  4. Преобразует запрос
    в голос и озвучивает

    Для преобразования текста
    в голос программа использует
    библиотеку SpeakerPy
